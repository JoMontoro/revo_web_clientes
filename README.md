# ♻️ REVO MATCH CIRCULAR

> Plataforma web comercial para conectar empresas con productos derivados de neumáticos fuera de uso (NFU), mostrando soluciones sostenibles y facilitando la solicitud de cotizaciones.

---

## 📌 Descripción

**REVO MATCH CIRCULAR** es un prototipo de página web comercial orientado a empresas interesadas en productos fabricados a partir de neumáticos fuera de uso (NFU).

La plataforma permite explorar un catálogo de productos, conocer sus aplicaciones comerciales, estimar el impacto ambiental de su utilización y solicitar propuestas comerciales mediante un formulario interactivo.

---

## 🎯 Objetivo

Desarrollar una plataforma web que permita:

- Atraer empresas interesadas en productos derivados de neumáticos fuera de uso.
- Mostrar aplicaciones comerciales según el sector empresarial.
- Facilitar el contacto mediante solicitudes de cotización.
- Promover la economía circular y la sostenibilidad.

---

## ✨ Características

- 📱 Landing Page comercial responsive.
- 🔍 Catálogo filtrable por sector.
- 📦 Fichas informativas de productos.
- 🤝 Formulario interactivo de matching.
- 📄 Simulación de propuesta comercial.
- 🌱 Calculadora de impacto ambiental.
- 💾 Captura de prospectos utilizando LocalStorage.
- ❓ Sección de preguntas frecuentes (FAQ).
- 🌐 Funcionamiento completamente offline (sin conexión a Internet).

## 🚀 Cómo ejecutar el proyecto

1. Descargar o clonar el repositorio.

```bash
git clone https://github.com/usuario/revo-match-circular.git
```

2. Ingresar a la carpeta del proyecto.

```bash
cd revo-match-circular
```

3. Abrir el archivo **index.html** con cualquiera de los siguientes navegadores:

- Google Chrome
- Microsoft Edge
- Mozilla Firefox

No es necesario instalar dependencias ni ejecutar servidores.

---

## 💻 Uso

1. Abrir la página principal (`index.html`).
2. Explorar el catálogo de productos.
3. Filtrar por el sector empresarial deseado.
4. Completar el formulario **"Cuéntanos qué necesitas"**.
5. Visualizar la simulación de propuesta comercial.
6. Consultar el impacto ambiental estimado.
7. Los datos ingresados quedarán almacenados en el **LocalStorage** del navegador.

## ⚠️ Limitaciones

Este proyecto corresponde únicamente al **Frontend**.

Actualmente **NO incluye**:

- Backend.
- Base de datos.
- Envío de correos.
- Autenticación de usuarios.
- Procesamiento de pagos.
- API REST.

Para convertir el prototipo en una aplicación de producción será necesario desarrollar un backend que gestione dichas funcionalidades.
